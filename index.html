what

This is a little demo showing what we had prepared for our spainjs talk.

Unfortunately our talk has not been accepted, we will try next year.

how

This demo is done with D3, a javascript library for data visualization, that shows how easy is to create a particle system with SVG.

It uses HTML5 audio API for realtime sound spectrum analysis, canvas and some lovely effects done with CSS3 animations.
